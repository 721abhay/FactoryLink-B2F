5. SECURITY REQUIREMENTS

Security is non-negotiable at every layer. FactoryLink handles real money, personal identity data, and
business financial information. Any breach destroys trust permanently.


LAYER 4 — DATA ENCRYPTION  |  AES-256 at rest. TLS 1.3 in transit. PCI DSS for payments.


LAYER 3 — AUTHORISATION  |  Role-based access. Factory cannot see customer data. JWT tokens.


LAYER 2 — AUTHENTICATION  |  OTP verification. Token refresh. Session management.


LAYER 1 — NETWORK  |  API Gateway rate limiting. HTTPS only. DDoS protection.
