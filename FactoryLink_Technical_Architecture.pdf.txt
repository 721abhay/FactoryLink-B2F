Layer 2 — API Gateway


FUNCTION

WHAT IT DOES

WHY IT MATTERS




Authentication

Every request carries a user token. Gateway
verifies token before passing request to
backend.

Prevents unauthorized access. Factory cannot
access customer data. Customer cannot see
factory internals.

Rate Limiting

Maximum requests per user per minute
enforced at gateway level.

Prevents any single user or bot from
overwhelming the system with requests.

Request
Routing

Gateway reads the request type and sends it
to the correct backend service.

Order requests go to Order Service. Payment
requests go to Payment Service. Never mixed up.

Logging

Every single request logged with timestamp,
user ID, action, and response time.

Essential for debugging problems and
understanding platform usage patterns.
